- Second item

  - Nested item 2
    - Deep nested item

### Ordered Lists

1. First step
2. Second step
3. Third step
   1. Sub-step A
   2. Sub-step B
4. Fourth step

### Task Lists

- [x] Complete task
- [ ] Incomplete task
- [ ] Another incomplete task
